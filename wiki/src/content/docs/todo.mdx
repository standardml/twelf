The priority now is working major content and/or visual regressions caused by the transition to MediaWiki. 

Steps:

- `<math>whatver</math>` needs to become `<Math formula="whatever"/>`
  if it's inline, and ` ```math ` displaymath otherwise.
- Guide links like `{{guide|chapter=2|section=4|title=Lexical Conventions}}` become
  `<Guide chapter="2" section="4">Lexical Conventions</Guide>`
- `{{keyword|key=.}}` becomes `<Keyword />` (the Keyword component figures out which one it is)
- Broken links become `<Todo>Broken link</Todo>`
- Remove `{{needs}}`, `{{discuss}}`, `{{case studies}}`, `{{undocumented}}` tags
- The generated footer can be deleted - it's just there to link to the web archive, but that's now a built-in part of the footer
- Pages should ideally get a `%%! description: "Blah blah blah"` since otherwise the search engines will just associate every page with the text "Home of the Twelf programming language" which is suboptimal
